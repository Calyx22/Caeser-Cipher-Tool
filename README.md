#Caeser Cipher Tool

A command-line encryption tool built in C as part of my CS50 journey.
It implements the the Caeser Cipher algorithm with three modes.

## Features
- Encrypts text with a custom key
- Decrypts text with a known key
- Brute force crack encryption text (tries all 25 keys)

## How to compile and run
make Tool
./Tool

## What I learned
- Caeser Cipher algorithm
- String manupilation in C
- Loops, functions and modular arithmetic
